This is a demonstration of the accessible dropdown (also known as a select-only combobox).

The dropdown for selecting the month should work according to the pattern and behaviour laid out in Select-Only Combobox Example | WAI-ARIA Authoring Practices 1.2 .

  • The dropdown is focusable with <tab>
  • The menu will open when <up>, <down>, <enter> or <space> are pressed when focused
  • The menu will open if the user starts typing

When open:

  • Pressing the <home> and <end> keys will highlight the first or last item respectively
  • Pressing the <up> or <down> keys will move the highlight accordingly
  • Pressing <enter> or <space> will close the menu, change the selected value to the current highlighted value, but keep focus on the dropdown
  • Pressing the <tab> key will close the menu, change the selected value, and move focus to the next element
  • Pressing the <escape> key will close the dropdown without changing its value

<option> elements work just like regular HTML Option elements:

The value attribute is optional, and a selected attribute can be used to set the initial value. 

<form id="form">
    <label id="label-for-dropdown">Choose a month</label>
    <dropdown-selector aria-labelledby="label-for-dropdown">
        <option value="0">January</option>
        <option value="1">February</option>
        <option value="2">March</option>
        <option value="3">April</option>
        <option value="4">May</option>
        <option value="5" selected>June</option>
        <option value="6">July</option>
        <option value="7">August</option>
        <option value="8">September</option>
        <option value="9">October</option>
        <option value="10">November</option>
        <option value="11">December</option>
    </dropdown-selector>
</form>
